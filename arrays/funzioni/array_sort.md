## Funzione `array_sort`

Ordina i valori in ordine crescente (usa _array_reverse_ per desc).

## Sintassi

array_sort(_array_) 

## Argomenti

* _array_ un array

## Esempi

* `array_sort(string_to_array( 'a,b,c,a,a')) → <array: 'a', 'a', 'a', 'b', 'c'>`
* `array_sort( array(3,4,5,3,2,1,14,14,1)) → <array: 1, 1, 2, 3, 3, 4, 5, 14, 14>`
